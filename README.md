# Traffic Light Controller using VHDL

## Overview
This project implements a Traffic Light Controller using VHDL for FPGA. The controller is designed using a Finite State Machine (FSM) and simulated using Xilinx Vivado.

## Features
- Finite State Machine (FSM) based design
- Red, Yellow and Green light sequencing
- VHDL implementation
- Simulation using Xilinx Vivado
- FPGA compatible design

## Tools Used
- VHDL
- Xilinx Vivado
- FPGA
